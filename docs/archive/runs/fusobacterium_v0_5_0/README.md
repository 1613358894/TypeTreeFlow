# Fusobacterium v0.5.0 Run Evidence

## Source Directories

- `results/fusobacterium_delivery_16of17`
- `results/fusobacterium_final_audit_v2`
- `results/fusobacterium_mortiferum_final_review`

## Purpose

This directory preserves a compact evidence set for the final `Fusobacterium`
v0.5.0 strict NCBI Assembly type-strain delivery state. It records the 16/17
delivery boundary, the final audit tables, and the final decision for the one
pending species, `Fusobacterium mortiferum`.

## Retained Files

- Markdown summaries and decisions from delivery, final audit, and final review
  outputs.
- TSV manifests and audit tables that identify selected assemblies, source
  output directories, evidence layers, completion status, and reviewed
  `F. mortiferum` candidates.

## Excluded Files

Large files and reproducible intermediates were intentionally not archived
here. This excludes genome FASTA/FNA files, 16S FASTA files, cache directories,
NCBI download artifacts, `barrnap` outputs, phylogeny/IQ-TREE outputs, and
other generated run products that can be regenerated or remain in their
original local run directories.
